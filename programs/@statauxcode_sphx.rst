.. code-block:: none

    Usage: @statauxcode code
    Returns name (or number) of statistics code
    Based on README.attributes:
          0 <--> value
          1 <--> obsolete
          2 <--> fico
          3 <--> fitt
          4 <--> fift
          5 <--> fizt
          6 <--> fict
          7 <--> fibt
          8 <--> fibn
          9 <--> figt
          10 <--> fipt
          11 <--> fbuc
    
    Example:
    @statauxcode fipt
    @statauxcode 4
