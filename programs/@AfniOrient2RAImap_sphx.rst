******************
@AfniOrient2RAImap
******************

.. code-block:: none

    Usage: @AfniOrient2RAImap <Orientation code> .....
    returns the index map fo the RAI directions
    
    examples:
    @AfniOrient2RAImap RAI
    returns: 1 2 3
    @AfniOrient2RAImap LSP
    returns: -1 -3 -2
